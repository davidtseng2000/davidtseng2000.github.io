Desktop Music Player

簡介（繁體中文版）

Desktop Music Player (DMP) 是一個桌面音樂播放器，使用 Python 和 PyQt5 來設計圖形使用者介面，並利用 VLC 來進行媒體播放。

功能

  • 播放、暫停、上一首、下一首、重複、隨機播放

  • 可調播放速度：可以更改播放速度（0.5x 至 2x）。

  • 音量控制：調整音量從 0 到 100。

  • 從資料夾載入歌曲

關於 Vibe Coding

此專案是為了練習 Vibe Coding 所做的，程式碼的生成主要是借助 ChatGPT。

立即使用

您可以在 GitHub 上查看完整的程式碼。

可自行修改本專案！




Introduction (English ver.)

This project is a Desktop Music Player created using Python and PyQt5 for the graphical user interface, while leveraging VLC for media playback. The main features of this player include:

  • Folder-based song loading
  • Next/previous track controls
  • Repeat and shuffle modes
  • Volume control
  • Adjustable playback speed
  • Attractive user interface design

Features

  • Play, Pause, Next, Previous, Repeat, Shuffle: Intuitive controls for music playback.
  • Adjustable Speed: You can change the playback speed (0.5x to 2x).
  • Volume Control: Easily adjust the volume from 0 to 100.
  • Folder-Based Song Loading: Load songs from a selected folder.
  • Attractive UI: User-friendly interface designed for better experience.

About Vibe Coding

This project was created as a Vibe Coding practice, a collaborative coding style where humans and AI work together creatively. The code was generated with the help of OpenAI’s ChatGPT, blending human ideas with AI-assisted code development.
